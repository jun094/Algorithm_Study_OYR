# Algorithm_study_OYR

Sejoing Univ. 웹개발 동아리 Open Year Round 알고리즘 스터디

-   1~2주차 네이버 공채 대비 문제 풀이 진행, 이후 주차부터 유형별 문제 풀이 진행

-   한문제 풀 때마다 commit / 매주 금요일 id_branch -> week_branch / 매주 일요일 week_barnch -> main_branch
